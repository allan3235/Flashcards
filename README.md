# Flashcards - Computo Bioinspirado

Aplicacion web estatica de estudio basada en flashcards para repasar conceptos de Computo Bioinspirado. Esta pensada para abrirse directamente en navegador y desplegarse facil en GitHub Pages.

## Que hace actualmente

- Muestra preguntas y respuestas en formato tarjeta.
- Permite voltear la tarjeta para ver la respuesta.
- Navega entre tarjetas con botones de anterior y siguiente.
- Mezcla el orden de las tarjetas con un boton de aleatorizacion.
- Muestra contador de progreso (tarjeta actual y total).
- Mantiene el codigo organizado por archivos separados:
	- `index.html` para estructura.
	- `styles.css` para estilos.
	- `script.js` para logica.
